 FEATURE 2: Hybrid Search with Reciprocal Rank Fusion (RRF)

FILE: src/hybrid_search_config.py (205 lines)
STATUS: 🎉 Complete redesign of retrieve.py fusion logic

BEFORE (Simple Weighted Average):
  Vector Score: 0.95 (very confident)
  BM25 Score: 0.10 (low)
  
  Fused (60/40): 0.95*0.6 + 0.10*0.4 = 0.61
  
  Problem: One high score dominates ❌

AFTER (Reciprocal Rank Fusion):
  Vector Rank: 2nd position    RRF: 1/(60+2) = 0.0161
  BM25 Rank: 10th position     RRF: 1/(60+10) = 0.0143
  
  Fused (avg): 0.0152 ✓
  
  Benefit: Rankings more important than scores
  Result: More stable, robust results ✓

HOW IT WORKS:
  ┌──────────────────────────────────────────┐
  │ Query: "Recent company changes"          │
  ├──────────────────────────────────────────┤
  │                                          │
  │  Vector Search          BM25 Search      │
  │  ╔════════════════╗    ╔════════════════╗│
  │  ║ 1. Document A  ║    ║ 1. Document C  ║│
  │  ║ 2. Document D  ║    ║ 2. Document A  ║│
  │  ║ 3. Document B  ║    ║ 3. Document E  ║│
  │  ╚════════════════╝    ╚════════════════╝│
  │         ↓                     ↓          │
  │         └──────→ RRF Fusion ←──┘         │
  │                    ↓                    │
  │  Final Ranking:    ↓                    │
  │  1. Document A (ranked high by both)   │
  │  2. Document D (good semantic match)   │
  │  3. Document C (good keyword match)    │
  │  4. Document B, E (single mention)     │
  │                                        │
  └──────────────────────────────────────────┘

FILE: src/faithfulness_check.py (263 lines)
STATUS: 🆕 Completely new feature

WHAT IT DETECTS:
  
  ✓ Hallucinations
    Answer: "The CEO was born in 1985"
    Documents: No mention of birth year
    → Flagged: Entity not grounded ❌
  
  ✓ Unsupported Claims
    Answer: "Sales increased by 50%"
    Documents: "Sales improved"
    → Flagged: Specific number not supported ❌
  
  ✓ Contradictions
    Answer: "The product is not available"
    Documents: "Product available in US"
    → Flagged: Contradiction detected ❌
  
  ✓ Supported Claims
    Answer: "The company was founded in 1995"
    Documents: "Founded in 1995"
    → OK: Fully grounded ✓

HOW IT WORKS:
  ┌─────────────────────────────────────────┐
  │ Answer: "Company founded 1995, now 50k  │
  │          employees, provides consulting"│
  ├─────────────────────────────────────────┤
  │                                         │
  │ 1️⃣  Token Overlap (25%)                 │
  │    "founded", "1995", "employees" ...  │
  │    → 90% tokens found in docs ✓ = 0.90 │
  │                                         │
  │ 2️⃣  Semantic Consistency (30%)          │
  │    Does answer match document MEANING?  │
  │    → High similarity ✓ = 0.92           │
  │                                         │
  │ 3️⃣  Entity Grounding (25%)              │
  │    "1995", "50k", "consulting"         │
  │    → 80% of specific claims in docs ✓   │
  │    = 0.80                              │
  │                                         │
  │ 4️⃣  Contradiction Detection (20%)       │
  │    Looking for logical inconsistencies │
  │    → None found ✓ = 1.0                │
  │                                         │
  ├─────────────────────────────────────────┤
  │ OVERALL SCORE:                          │
  │ 0.25*0.90 + 0.30*0.92 + 0.25*0.80 +    │
  │ 0.20*1.0 = 0.89 ✓ (HIGH)              │
  └─────────────────────────────────────────┘

SCORE INTERPRETATION:
  
  🟢 0.9-1.0  EXCELLENT
     Answer fully grounded in documents
     Confidence: Very High
  
  🟢 0.75-0.9 GOOD
     Minor issues or missing details
     Confidence: High
  
  🟡 0.6-0.75 ACCEPTABLE
     Some concerns, may need review
     Confidence: Moderate
  
  🔴 <0.6    POOR
     Likely hallucinations
     Confidence: Very Low
